# 爱拼安小匠的 FreeCodeCamp Tribute

## 任务
完成一个简易的弹幕墙。要求：
 - 允许用户即时发送弹幕
 - 可清除屏幕上显示的所有弹幕

## 功能特性
 - 满足基本的弹幕发送需求
 - 每条弹幕显示完毕后，其对应的DOM元素就会被自动清除
 - 每条弹幕的颜色、字体大小与垂直出现位置都是不一样的
 - 弹幕实现的核心过程写在专门的JS中，便于以后强化

## 用到的技术
- jQuery
- [Materialize CSS 配色方案](http://materializecss.com/color.html)
